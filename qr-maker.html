QR Code 產生器

貼上網址或文字，即時生成，一鍵下載

衛教單張想附連結、課程報名表要放上投影片、名片想加聯絡資訊 — 這些場合都需要一個 QR Code。網路上多數線上工具要嘛塞滿廣告，要嘛要求註冊帳號才能下載高解析度圖檔。這個工具只做一件事：貼上內容、選好尺寸，馬上生成可下載的 QR Code，不需要帳號、不需要安裝任何軟體。

🔒 內容不會離開你的電腦：QR Code 的生成運算全部在你的瀏覽器內完成，貼上的網址或文字不會上傳到任何伺服器。含個案資訊的內容也請留意 — 建議只放公開連結，不要把病人資料編進 QR Code 內容。

常見使用場景

📋 衛教單張連結

單張紙印不下完整的衛教內容，QR Code 掃了就能連到完整版網頁 — 版面不擠，資訊也不打折扣。

📝 Google 表單報名

把課程報名表單網址轉成 QR Code，貼在公告欄或群組海報上，學員掃描即可直接填寫，不必手動輸入落落長的網址。

🖥️ 投影片最後一頁

演講、實體課投影片的最後一頁放上主辦單位網站或問卷連結的 QR Code，聽眾拍照就能收藏，比口頭念網址有效得多。

💼 名片與聯絡資訊

把個人網站、預約掛號連結，或本篇下方提到的 Wi-Fi 格式編成 QR Code，印在名片或診間牆上，來訪者掃描即可取得。

常見問題

❓ 掃不到怎麼辦

優先檢查三件事：QR Code 與背景是否對比足夠（深色圖案配淺色底最穩，避免套用花俏色彩）；周圍是否留有足夠白邊（本工具已自動加上）；列印尺寸是否過小 — 手機鏡頭對太小的 QR Code 較難對焦，海報用途建議選 1024px 尺寸。

❓ 內容改了，QR Code 要重印嗎

如果 QR Code 裡放的是網址（例如公告頁面連結），只要那個網頁本身的內容更新，QR Code 完全不必重印，掃描後看到的就是最新內容。但如果日後網址本身改變（例如換了網域），舊的 QR Code 就會失效，需要重新生成。

❓ 可以放 Wi-Fi 密碼嗎

可以。在輸入框貼上這個格式：WIFI:T:WPA;S:名稱;P:密碼;;（把「名稱」「密碼」換成實際的 SSID 與密碼），大多數手機掃描後會直接提示連線，不必手動輸入。

🎯 現在就做：把你下一份衛教單張或報名表單的連結貼進上面的輸入框，選好尺寸與容錯等級後下載 PNG，直接插進 Word 或簡報裡試試看 — 用手機相機掃一次，確認真的能開啟正確的頁面。
