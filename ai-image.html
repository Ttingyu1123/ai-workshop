觀念充電 — AI 是怎麼「畫」出圖的？

不需要懂數學，但知道 AI 生圖的基本原理，很多日常現象會突然說得通 — 為什麼同一句 Prompt 每次生的圖都不一樣？為什麼 AI 常把字畫壞？目前主流生圖模型分兩大流派：

流派一：擴散模型（Diffusion）— 從雜訊中「顯影」

Stable Diffusion、Midjourney、FLUX 這些經典生圖模型，用的都是「擴散」原理：

  • 訓練時 — 把幾億張圖片一步步加上雜訊直到面目全非，讓 AI 學會「逆著走」：從雜訊還原出圖片
  • 生圖時 — 從一張純雜訊開始，AI 依照你的文字描述一步步去除雜訊，圖片逐漸浮現
  • 起點是隨機的 — 每次的起始雜訊（種子，seed）不同，所以同一句 Prompt 每次生出的圖都不一樣
就像 X 光片在顯影液中逐漸浮現影像 — 擴散模型的每一步去噪，都是讓圖片從一片雪花雜訊中「顯影」出來。每次沖片的起點不同，成品就有差異 — 所以「抽卡」（多生幾張挑最好）是合理策略，不是你 Prompt 寫得不好。

流派二：原生多模態 — 像寫文章一樣「寫」出圖

ChatGPT（GPT Image 2）和 Gemini（Nano Banana）屬於新一代做法：圖像和文字在同一個模型腦袋裡處理，生圖時像寫文章一樣，一小塊一小塊地「寫」出來。

  • 為什麼它們寫得出中文字 — 對這類模型，文字是「語言」不是「形狀」，所以能寫出正確的字（詳見下方「中文字生成新時代」）
  • 為什麼它們聽得懂修改指令 — 生圖和對話是同一個腦袋，「把背景改成藍色」它真的理解
  • 代價 — 生圖速度較慢，藝術風格的極致表現通常不如 Midjourney 這類專精模型
一句話總結：擴散模型是「雕刻家」— 從雜訊石塊裡雕出圖，藝術感強但對「寫字」不拿手（尤其中文）；原生多模態是「作家」— 用寫的做出圖，識字、聽得懂話，但風格極致度略遜。知道手上工具是哪一派，就知道該期待它什麼。

Prompt 四要素公式

生圖 Prompt 更像是填表格 — 把四個要素寫清楚就好：

1. 主題 — 畫什麼內容？

Good：一位穿白袍的女醫師正在跟老年病人解釋藥物用法

Bad：醫院的圖

2. 風格 — 什麼視覺風格？

Good：扁平設計插畫，線條簡潔，色彩柔和明亮

Bad：好看的風格

風格詞彙：扁平插畫、卡通、水彩、日系、極簡、infographic、醫學期刊風、3D 渲染

3. 構圖 — 怎麼排版？

Good：正方形比例，白色背景，主角置中，周圍留白

Bad：（沒寫）

指定比例（正方形/4:5/16:9）和背景色。IG 用 4:5，簡報用 16:9。

4. 限制 — 不要什麼？

Good：不要出現文字，不要寫實風格，不要血腥畫面

Bad：（沒寫）

特別重要：「不要文字」（除非用 GPT Image 2 或 Nano Banana Pro）。

衛教圖 Prompt 範例

以下每個 Prompt 都可以直接複製貼到 ChatGPT 中使用。點右上角按鈕一鍵複製。每個 Prompt 下方附上同一段 Prompt 分別丟給 ChatGPT 和 Gemini 的實際生成結果，讓你比較兩個工具的表現差異 — 點縮圖可放大檢視完整圖片。

基礎：單張衛教插圖

衛教插圖 — 洗手步驟
請幫我生成一張衛教插圖。 主題：正確洗手的七個步驟 風格：扁平設計插畫，色彩柔和明亮，適合醫療場所張貼 構圖：直式，每個步驟一格，共七格排列 限制：不要出現文字（我會自己加），不要寫實風格

進階：社群衛教圖卡

IG 衛教圖 — 糖尿病飲食
生成一張 Instagram 衛教圖（4:5 比例）。 主題：糖尿病患者的飲食原則 — 三多三少 風格：清新簡約的資訊圖表風格，莫蘭迪配色（淡紫、奶油白、薄荷綠） 構圖：上方大標題區，中間分成兩欄（推薦食物 vs 避免食物），下方放一行小字版權 限制：食物用可愛的插畫圖示表示，不要放真實食物照片。圖上不要出現任何文字，我會用 Canva 自己加。

學術：簡報概念圖

學術示意圖 — 抗藥性傳播
生成一張學術簡報用的概念示意圖。 主題：抗生素抗藥性的傳播途徑（人→動物→環境→人的循環） 風格：專業醫學期刊插圖風格，配色以深藍、白、淺灰為主 構圖：圓形循環圖，四個節點用箭頭連接，每個節點有一個簡單 icon 限制：不要卡通風格，要能放在正式學術簡報中。不要出現文字，我會自己標註。

素材用：背景/圖示

背景素材 — 醫療主題
生成一張可以當衛教單張背景的圖案。 內容：淡淡的醫療元素圖案散布在畫面中（聽診器、藥丸、心跳線、十字符號） 風格：線條簡筆畫，單色淺灰色，極淡 構圖：A4 直式比例，圖案均勻分布不要太密 用途：這張圖會被用作背景，上面會疊加文字，所以圖案要很淡、不能搶眼
小圖示組 — 衛教用
生成一組 8 個醫療相關的小圖示（icon set）。 內容：洗手、量血壓、吃藥、打針、戴口罩、量體溫、運動、喝水 風格：圓形圖示，扁平設計，統一使用柔和的藍綠色調 構圖：2 行 4 列排列，每個圖示之間有足夠間距 限制：每個圖示風格要完全一致。白色背景。不要文字。

生圖進階技巧

1. AI 不會畫中文字（但有例外）

大部分 AI 生成的圖上如果有中文字，幾乎 100% 是亂碼或錯字。通用解法：在 Prompt 中寫「不要出現任何文字」，然後用 Canva 或 PowerPoint 自己加。

例外：GPT Image 2 和 Nano Banana Pro 已經能正確渲染繁體中文字。詳見下一章「中文字生成新時代」。

2. 英文 Prompt 效果更好

模型訓練資料以英文為主，英文 Prompt 對風格的控制力更強。不會寫？叫 AI 幫你翻：

中翻英 Prompt 助手
我想用 AI 生成一張圖片，以下是我的中文描述，請幫我翻譯成適合 AI 生圖的英文 prompt。翻譯時加入更多視覺細節和風格描述詞彙： [在這裡貼上你的中文描述]

3. 多生幾張挑最好的（俗稱「抽卡」）

同一個 Prompt 生三次，三張圖會不一樣。不要期待一次到位 — 生 3-4 張，挑最好的。這個過程俗稱「抽卡」，跟手遊轉蛋一樣，多抽幾次才容易中獎。

4. 對話式修圖

ChatGPT 可以用對話修改已生成的圖，不需要重新描述整張圖：

注意：對話式修圖的缺點是圖片會愈改愈模糊。每次修改 AI 都會重新生成整張圖，經過多輪修改後畫質會明顯下降。建議修改不超過 2-3 輪，如果不滿意就用完整的新 Prompt 重新生成。
修圖指令範例
請修改這張圖： 1. 把左上角的人物換成穿白袍的醫師 2. 背景色改成淡藍色 3. 保持其他所有元素不變

5. 維持風格一致性

做系列衛教圖時，每個 Prompt 重複描述相同的風格參數：

系列圖 — 維持風格一致
請用跟上一張圖完全相同的風格生成一張新圖。 需要保持一致的元素： - 相同的插畫風格（扁平設計、圓角造型） - 相同的配色（主色淡紫 #A186B4、底色奶油白 #F8F7EE、輔色薄荷綠 #E3F3F1） - 相同的線條粗細和圖示設計 - 相同的構圖比例（4:5） 新圖主題：高血壓患者的日常注意事項

實測：先生成「高血壓」主題，再用上面的指令要求「同風格、換主題」生成第二張 — 可以看到兩張圖的配色、線條、版型都維持一致：

6. 用參考圖片

如果你有一張喜歡的風格參考圖，直接上傳給 ChatGPT 說「用這個風格畫一張新的…」。AI 會分析參考圖的風格元素（配色、線條、構圖），比用文字描述更精準。

Infographic 與複雜衛教圖

除了簡單的插圖，AI 現在也能生成結構較複雜的 Infographic — 包含數據、流程圖、對比圖等。這類圖在衛教上特別實用。

先講清楚分工：臨床數字、用藥資訊與警示徵兆必須由你提供並核對來源，寫進 Prompt 讓 AI 排版 — AI 只負責版面與視覺化，不負責醫療內容的正確性。讓 AI 自己「補」盛行率或用藥時程，就是把幻覺印在海報上。

Infographic Prompt 結構

複雜圖比簡單插圖需要更詳細的 Prompt。關鍵是把版面結構寫清楚：

Infographic — 流感 vs 感冒比較
生成一張醫療 infographic 海報。 主題：流感 vs 普通感冒的差異比較 版面結構： - 最上方：大標題區（留空，我會加字） - 中間：左右兩欄對比（左=流感，右=感冒） - 對比項目：發燒程度、症狀出現速度、全身痠痛、疲倦程度、併發症風險 - 每個項目用圖示+色彩深淺表示嚴重程度 - 最下方：「何時該就醫」的三個警示徵兆 風格：清爽的醫療 infographic，用藍色(流感)和綠色(感冒)區分兩側 構圖：直式 A4 比例，適合列印張貼 限制：圖示化為主，文字部分只放英文（我會替換成中文）
Infographic — 用藥時間軸
生成一張時間軸風格的衛教 infographic。 主題：糖尿病患者的一日用藥時程 版面結構： - 橫式時間軸從左到右（早晨 → 中午 → 晚上 → 睡前） - 每個時段： - 時鐘圖示 - 該時段的用藥/檢測動作（用圖示表示） - 飲食建議圖示 - 底部：注意事項區（3 個警示圖示） 風格：柔和的莫蘭迪色系，圓角卡片設計 構圖：橫式 16:9，適合簡報投影 限制：不要出現中文字，只用圖示和簡單英文標記
Infographic 的最佳實踐：先用 AI 生成圖示化的版面框架，再用 Canva 疊加繁體中文文字。這樣既能利用 AI 的設計能力，又能確保中文字的清晰度。詳見 番外 — Canva 設計入門。

中文字生成新時代

2025 年之前，AI 生圖最大的痛點就是「不會寫中文字」。現在 GPT Image 2 和 Nano Banana Pro 已經大幅改善 — 短標題和少量文字可用，長段內文仍會出錯（什麼情境能用，見下方對照）。

什麼時候該讓 AI 直接寫字？

✓ 適合 AI 寫字的情境
  • 衛教海報的大標題（2-8 個字）
  • 社群圖卡的金句
  • 簡報封面標題
  • 數字資訊（統計數據、百分比）
✕ 不建議的情境
  • 大量內文（容易出錯）
  • 藥名、專業術語（錯字風險高）
  • 需要精確排版的表格
  • 法規要求的免責聲明文字

含中文字的 Prompt 技巧

含標題文字的衛教海報
生成一張衛教海報。 主題：預防流感 圖片上的文字： - 大標題：「預防流感 從手開始」（置頂，大字，白色） - 副標：「正確洗手 5 步驟」（標題下方，較小） 風格：扁平設計，主色調淡藍色，明亮溫暖 構圖：直式 3:4，標題在上方 1/3，下方 2/3 是洗手步驟插圖 注意：繁體中文字要清晰可讀，字體風格要統一
想像 AI 寫中文字就像一個外國人練書法 — GPT Image 2 已經像是練了三年的外國人，大字寫得不錯，但長篇內文還是會出錯。所以大標題讓 AI 寫，小字自己加。

  • 額度制，大量使用需付費
想體驗 ComfyUI 生態但沒有高階顯示卡，這是門檻最低的入口。同類服務還有官方的 Comfy Cloud（需訂閱）、Civitai 線上生成等，概念相同。操作步驟見 本機／開源生圖入門 的路線 B。
怎麼選：九成衛教需求 → ChatGPT / Gemini（免費、識字、聽得懂話）。追求視覺美感 → Midjourney。在意隱私或想深度客製 → ComfyUI（需要顯示卡）。想試開源生態但沒有顯示卡 → RunningHub。

醫療圖像的倫理紅線

✕ AI 生圖不能做

  • 上傳真實病人照片、醫學影像、病歷截圖到雲端生圖工具 — 除非有機構核准、合法授權與資料處理條款
  • 偽造醫學影像 — X 光、MRI、CT、病理切片
  • 生成特定病人肖像
  • 製作假的前後對比圖
  • 未標註 AI 生成用於學術
  • 誤導性醫療宣傳

✓ 適合的用途

  • 概念性衛教插圖
  • 社群媒體圖卡
  • 簡報視覺輔助
  • 海報與宣傳品
  • 教學教材
  • 圖示與 icon

原則：AI 生成的圖適合用在「概念示意」，不適合用在「呈現事實」。
