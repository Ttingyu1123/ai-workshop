番外・小工具

Prompt 產生器

選一選、填一填，好 Prompt 直接複製帶走

不確定 Prompt 怎麼下？從這裡開始。選好產出類型、主題和對象，下面就會即時組出一段完整的 Prompt — 複製後貼到 ChatGPT、Gemini 或 Google Flow 就能用。這個工具完全在你的瀏覽器裡運作，不連任何 AI、不上傳任何資料。

⚠️ 主題與補充內容請勿包含任何病人可識別資訊（見 Part 3）
詳細版會把你選的風格、對象、用途展開成具體規格 — 第一次生成就更接近你要的；標準版留比較多空間給後續對話微調。
✨ 你的 Prompt（即時更新）
產生的 Prompt 只是起點：貼過去之後，第一版結果通常不會完美 — 用 Part 2 教的迭代追問繼續調整（「第二點再簡單一點」「改成直式構圖」），不要整個重來。生圖類 Prompt 建議搭配 Part 4 的觀念服用。
