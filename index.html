🧰 番外工具

獨立成篇的工具教學 — 需要哪個看哪個。

番外・小工具

Prompt 產生器

填空就有好 Prompt
選主題、對象、風格，自動組出可直接複製的 Prompt。純網頁工具，不用帳號、不連 AI，離線也能用。
番外・瀏覽器工具

Chrome Skills

網頁版的 AI 快捷鍵
把常用的提示詞存成技能，打斜線一鍵叫出。綁定當前網頁與分頁，讀論文、查藥、做衛教最順手。附醫療人員實用範本。
番外・文獻工具

NotebookLM 文獻閱讀

只根據你給的資料回答
上傳 guideline 和 paper，提問時每句回答都附出處。語音摘要通勤聽，讀書會、晨會準備的神器。
番外・語音工具

語音轉文字

錄音變逐字稿
會議、演講、讀書會錄音一鍵轉文字，再交給 AI 整理成會議紀錄與重點摘要。
番外・剪輯工具

CapCut 剪輯實戰

衛教影片的最後一哩路
AI 生成的素材最後都要進剪輯軟體組裝。時間軸、自動字幕、配樂到匯出，一支手機就能完成。
番外・設計工具

Canva 設計入門

排版加字的好幫手
AI 生圖的中文字常出錯？用 Canva 模板排版、疊加繁中文字，衛教單張、簡報、社群圖卡一次搞定。
番外・建站教學

用 GitHub Pages 架網站

從零開始的建站教學
這個網站本身就是用 GitHub Pages 架的！從 HTML 基礎到部署上線，手把手帶你做出第一個網站。
番外・本機生圖

本機／開源生圖入門

資料不出機器的生圖法
用 ComfyUI 在自己電腦上生圖，輸入完全不上雲、免費無限次。沒有獨立顯示卡？用 RunningHub 線上跑一樣可以。
